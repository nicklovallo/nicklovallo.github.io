Favorite Movies


Tech Used:

Python, Flask, PostgreSQL, Heroku, Bootstrap, HTML


Overview:

Full-featured website where users can create an account and log on, create their list of favorite movies, and view the lists of their friends.


Difficulties Encountered and Skills Learned:

I spent a significant amount of time on this project thinking about what I wanted it to be and what features I should include. I eventually decided to include an account/login feature, so that users could interact with their own movie list posting with full CRUD functionality. Through working on this project, I learned a wealth of skills relating to the Flask framework for Python, working with a SQL database, and deployment best practices.
